# Memes Gallery
Приложение для просмотра, поиска, создания и управления мемами.

## Функционал
- Просмотр популярных мемов с пагинацией (12 мемов на странице).
- Лайк / разлайк мемов и просмотр избранных.
- Поиск мемов по названию.
- Создание и удаление мемов.
- Редактирование названия и изображения мема.

## Технологии
- React + TypeScript
- Redux Toolkit
- React Router
- React Hook Form
- Mantine UI
- Vite
